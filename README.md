# 🌸 Iris Flower Classification

## 📌 Project Description
This project is developed as part of the CodeAlpha Data Analytics Internship.

The project uses Machine Learning to classify Iris flowers into three species:
- Setosa
- Versicolor
- Virginica

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📊 Features
- Data Loading
- Data Analysis
- Data Visualization
- Decision Tree Classification
- Confusion Matrix
- Classification Report
- Prediction System
